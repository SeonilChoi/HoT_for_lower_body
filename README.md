# HoT for Lower Body

This repository is a fork of [Hourglass Tokenizer (HoT)](https://arxiv.org/pdf/2311.12028) for Efficient Transformer-Based 3D Human Pose Estimation (CVPR 2024), modified to perform 3D pose estimation for the **lower body only**.

Using the original HoT code as a reference, this project is set up to detect and estimate **lower body joints only**, rather than the full body.

## Reference

- [HoT: Hourglass Tokenizer for Efficient Transformer-Based 3D Human Pose Estimation](https://github.com/NationalGAILab/HoT) (CVPR 2024)
